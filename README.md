node-webapp-boilerplate
=======================

Boilerplate for node web applications.

## Usage
* git clone https://github.com/alyoshakiss/node-webapp-boilerplate.git
* npm install
* http://localhost:3001/

## Components

### Frontend

* Zurb Foundation 4
* Jade Templating Engine
* JQuery

### Backend

* Node.js
* Express
* Winston
